## ❓ FAQ RÁPIDO

### P: "Como sei qual enquadramento?"
**R:** Verifique a renda bruta anual do associado:
- Até R$ 500 mil = PRONAF
- R$ 500 mil a R$ 3,5 milhões = PRONAMP
- Acima de R$ 3,5 milhões = Agricultura Empresarial

### P: "Precisa de documentos agora?"
**R:** Não! O sistema é apenas consultivo. Use para informar cliente quais linhas existem.

### P: "Posso emprestar múltiplas linhas?"
**R:** Geralmente não. Uma linha de crédito por vez. Após pagar, pode solicitar outra.

### P: "Quanto tempo demora?"
**R:** Sistema busca em segundos. Análise de crédito leva 5-15 dias úteis.

### P: "Taxa pode mudar?"
**R:** Sim! Conforme Selic. As taxas mostradas são as vigentes em junho/2026.

### P: "É aprovado automaticamente?"
**R:** Não. Sistema mostra quais linhas existem. Análise de crédito é feita depois.

### P: "Pode desistir depois?"
**R:** Sim. Sem penalidade se não usar o crédito dentro do prazo permitido.

### P: "Vale para todos os estados?"
**R:** A maioria sim. PRONAF/PRONAMP são nacionais. Verificar instituição responsável.

### P: "Qual é a melhor linha?"
**R:** Depende! Cada uma é melhor para uma situação:
- **Menor juros**: PRONAF Agroecologia (0,5%)
- **Maior prazo**: Moderagro (12 anos)
- **Menor burocracia**: PRONAF B
- **Maior valor**: Moderfrota/PCA
