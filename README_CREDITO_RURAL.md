# 🌾 SISTEMA DE CONSULTA DE CRÉDITO RURAL - CRESOL

## 📦 O QUE FOI ENTREGUE

Sistema completo de consulta de linhas de crédito rural para a Cresol, desenvolvido em Google Apps Script com interface web, motor de regras inteligente, painel administrativo e histórico de auditoria.

---

## 🔐 SEGURANÇA

### Dados Armazenados
✓ Data/hora das consultas  
✓ Parâmetros buscados (sem dados do cliente)  
✓ E-mail do consultor (auditoria)

### Dados NÃO Armazenados
❌ Nomes de clientes  
❌ CPF ou dados pessoais  
❌ Valores de crédito  
❌ Histórico financeiro

### Conformidade
✓ LGPD (sem dados sensíveis)  
✓ Sigilo bancário (dados agregados)  
✓ Auditoria completa  
✓ Backup automático

## ❓ PERGUNTAS FREQUENTES

### P: Sistema é grátis?
**R:** Sim! Google Apps Script e Google Sheets são gratuitos. Custo é apenas em treinamento/customização.

### P: Posso usar em celular?
**R:** Sim! Sistema é responsivo. Melhor em desktop/tablet, mas funciona em celular.

### P: Quanto tempo leva para implementar?
**R:** 5-7 dias de trabalho (2-3 horas/dia).

### P: Preciso atualizar as linhas?
**R:** Sim. Recomenda-se revisar mensalmente (taxas podem mudar com a Selic).

### P: Funciona offline?
**R:** Não. Precisa de conexão internet (usa Google Sheets).

### P: Posso usar em várias agências?
**R:** Sim! Uma única planilha para toda a Cresol. Cada colaborador tem seu acesso.
